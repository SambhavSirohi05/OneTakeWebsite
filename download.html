⚠️ Gatekeeper Exception Steps

Since OneTake is independently developed, macOS might show an "unidentified developer" warning. To run it, run this command in Terminal:

xattr -cr /Applications/OneTake.app

Then right-click OneTake in your Applications folder and click Open. This only needs to be done once!
