Gatekeeper Exception Steps

Since OneTake is independently developed, macOS might show an "unidentified developer" warning. To run it, copy and run this terminal command:

xattr -cr /Applications/OneTake.app

After executing the command, right-click OneTake in your Applications folder and click Open. This only needs to be done once!

BUILD FROM SOURCE 
# Clone the official repository
git clone https://github.com/SambhavSirohi05/OneTake.git
